JavaScript Novice to Ninja:

Chapter 9: The Window Object

BOM (Browser Object Model) - collection of properties and methods that contain browser and computer screen information

window - global obj in browser

parseInt, isNaN - methods of Number object

Methods:
alert, confirm() [true: OK; false: cancel], prompt() [true: OK; null: cancel]

window.navigator - info about user's browser and OS; can be unreliable

window.location - URL of current page

window.history - keeps info about visited pages in the session

window.screen - info about user's screen

document.write() - old; writing text to doc

document.cookie - store small pieces of info between requests

window.setTimeout() - invoke a function after a set amount of time

clearTimeout() - cancel setTimeout()

window.setInterval() - repeatedly invoke a function

clearInterval() - stop setInterval()

window.requestAnimationFrame() - optimized animation by using browser's built-in graphics

cancelAnimationFrame() - cancel requestAnimationFrame()

Chapter 14: HTML5 APIs

data-attributes - embed custom data into webpage

Web Storage API - key-value pairs stored in user's device

Geolocation API - access to geographic coordinates of user's device

Web Worker API - perform computationally intensive tasks in the background

Websockets - allow real-time, two-way communication

Notification API - display notifications on the user's system

Chapter 5 - Objects

Objects - key-value pairs inside {}; mutable; can be used as parameters to functions;  copied by reference
Object Method - function property inside object

> Creating Objects:
Object literal 
const spiderman = {}

Constructor function
const spiderman = new Object(); 

Shorthand - if property key same as property value
const ironMAn = {name, realName}

> Accessing Objects
Access Objects: (dot notation or bracket)
Object.property or Object['property']

> Calling Methods
Object.method() or Object['method']()

> Check of Properties/Methods Exist

* Returns true even if property inherited from another object:
'property' in object
object.property !== undefined

* Only return property that belongs that particular object:
object.hasOwnProperty('property')

* Find all Properties of an Object
for (const key in object){
    console.log(key + ": " + object[key]);
}

* Only own properties returned:
for (const key in object){
if (object.hasOwnProperty(key)){
    console.log(key + ": " + object[key]);
}
}

Object.keys() - returns an array of all the keys of any object provided as an argument
for(const key of Object.keys(superman)){
console.log(key);
}
<< name
real name
height...

Object.values() - array of all object's value
for(cons value of Object.values(superman)){
console.log(value);
}

<< Superman
Clark Kent
75

Object.entries() - returns key-value pairs array
for(const [key,value] of Object.entries(superman)){
console.log(`${key}: ${value}`);
}

name: Superman
real name: Clark KEnt
height: 75

> Adding Properties
superman.city = 'Metropolis'

> Changing Properties
superman['real name'] = 'Kal-El';

> Removing Properties
delete superman.fly

object within objects

> Built-in Objects
JSON - Javascript Object Notation; functions not permitted
const batman = '{"name": "Batman"}'

JSON.parse() - JSON format to JavaScript object
JSON.stringify() - JS Object to JSON format

Math object - for calculations or constants
Math.abs() - aboslute +value of number
Math.ceil() - rounds up to next int or same if already int (4.2 << 5)
Math.floor() - round down to next int or same if already int (4.2 << 4)
Math.round() - round to nearest int (4.499 << 4)
Math.trunc() - truncated at decimal point (4.9 << 4)
Math.random() - generate number between 0 (inclusive) and 1 (exclusive)
to generate bet 0 and specific num: num * Math.random()
to generate random int: Math.floor(num * Math.random())

Date object - create date objects; getter, setter methods
Regex - create regular expressions
any / /
range [a-z] [A-Z]
/[^A-Z]/ : not a capital letter
g global
i ignoreCase
m multiline

* Special Characters (See page 235)
.(any char except line breaks) 
\w(any word char) 
\W(any non-word char) 
\d(any digit) 
\D(any non-digit) 
\s(any white)

* Modifiers (see page 236)
? optional
*  or + match one or more occurences of preceding token

.split() .match() .search() .replace()

Chapter 7 - Events

Events - user interaction with webpage
Event listener - addEventListener('click', doSomething)
Event target - reference to node that fired the event
Event coordinates - screenX screenY, clientX clientY(usually browser window), pageX pageY (document)

Event types:
Mouse events - click, dblclick, mouseover
Keyboard events - keydown (key pressed/held down), keypress (keys that produce input), keyup (key release);

Modifier keys(Shift, Ctrl, Alt, Meta/Cmd in Mac - keydown/up but not keypress as no character produced)
Touch events - touchstart/end, touchmove, touchenter, touchleave, touchcancel

removeEventListener()

preventDefault()

Event propagation - order of events fired on element (Capturing, Bubbling)

Event delegation - event listener in parent element capture events to children elements

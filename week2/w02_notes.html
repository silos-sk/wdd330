Chapter 4 - Functions

Function literals defined by:
function declaration - function name(parameter){} - can be invoked before defined as they are hoisted on top o the scope
function expression - let variable = function(){} - can only be invoked after they are defined

name(argument)
rest operator - access multiple arguments as an array
arrow functions - const square = x => x*x
callback - function name(argument1, function) - function provided as an argument
DRY - Don't Repeat Yourself

Array Iterators:
map() - [1,2,3].map(square) iterates over an array and takes a callback function as a parameter that is invoked on each item in the array

reduce() - [1,2,3,4,5].reduce((acc,val) => prev + val); - cumulatively combines each result to return just a single value

filter() - numbers.filter (x => x%2 === 0) - returns a new array that only contains items from he original array that return true when passed to the callback

Chapter 8 - CSS Transforms and Transitions

> Transform: 
- only work on block elements
- second transform will override the first; order matters

Types:
- translate - move elements left, right, up or down.
transform: translate(x,y)
transform: translateX(45px)
transform: translateY(-30px)
- rotate - rotate around point of origin i.e. center
can be grads, radians, turns, degrees
transform: rotate(10deg);
- scale(x, y) - scales horizontally then vertically. 
One value = used for both x and y values. scale(1) = element same size; scale(2) double proportion
No impact on document flow
transform: scale( 1.5, 0.25) or scaleX(x) or scaleY(y)
- skew(x, y) - specify skew along x and y axis
transform: skew(15deg, 4deg)

Changing origin:
transform-origin: 0 0;

> Transitions:
- original state, final state, transition functions
- transition properties: 
transition-property: transform, color;  - defines CSS properties of the element that should be transitioned, 'all' default
transition-duration: 0.2s (optimum time) - how long transition will take; 
transition-timing-function - control the pace of the transition - ease, linear, ease-in, ease-out, or ease-in-out 
transition-delay - delay before transition begins

* Transition shorthand:
transition: transform 0.2s ease-out 50ms (duration must precede delay)

* Multiple transition:
transition: transform 0.2s ease-out 50ms, color 0.1s linear 50ms;
transition: all 0.2s ease-out 50ms

> Animations:
- Keyframes
@keyframes moveRight {
   from {
    transform: translateX(-50%);
} 
to {
transform: translateX(50%);
}
}

- Animation Properties:
animation-name: appearDisappear
animation-duration: 300ms
animation-timing-function: ease/ease-in/linear/ease-out/ease-in-out or steps(4, end)
animation-iteration-count: infinite
animation-direction: alternate / alternate-reverse
animation-delay: 50ms;
animation-fill-mode: backwards / forwards / both / none
animation-play-state: running / paused

- Shorthand Animation:
animation: 300ms ease-in alternate 5s backwards appearDisappear;

* Variations on fillStyle
CanvasPattern : createPattern(image to create pattern, how image should be repeated)

repeat value: repeat, repeat-x repeat-y, no-repeat

function drawPattern(){
   var canvas = document.getElementById("demo2");
   var context = canvas.getContext("2d");
   context.strokeStyle = "red"
  
   var img = new Image();
   img.src = "../images/bg-bike.png";
   img.onload = function(){ // create pattern once image is fully loaded by the browser
    var pattern = context.createPattern(img, "repeat");
    context.fillStyle = pattern;
    context.fillRect(10, 10, 100, 100);
    context.strokeRect(10, 10, 100, 100);
   };
}

* Drawing Other Shapes by Creating Paths
Paths - blueprint for lines, arcs, and shapes

function drawCircle(canvas) {
   var canvas = document.getElementById("myCanvas");
   var context = canvas.getContext("2d");
   context.beginPath();
   context.arc(50, 50, 30, 0, Math.PI*2, true);
   context.closePath();
   context.strokeStyle = "red";
   context.fillStyle = "blue";
   context.lineWidth = 3;
   context.fill();
   context.stroke();
}

arc(x, y, radius, startAngle, endAngle, anticlockwise)
x, y : where you want the arc's path to begin
startAngle, endAngle : start and end angles along the circle's circumference that you want to draw
arc angle: radian; circle 2PI
anti-clockwise: optional; true; otherwise clockwise : false

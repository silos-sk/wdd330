Chapter 11 - Further Functions

call(), apply() methods
IIFE (Immediately Invoked Function Expressions): (function(){}())
Recursive function - keep invoking itself until a certain condition is met
Callback - function provided as an argument to another function
Promises - nicer mechanism for multiple async actions in sequence
Higher-order functions - functions that return other functions
Closure - process of keeping a reference to a variable available outside the scope of the function it was originally defined in
Generator - function *; returns interator object that provides a next() method
Functional programming - breaking processes to steps in a series of functions
Pure functions - function independent from code state; always give the same result when given same args (referential transparency)
Currying - partial application; applying one argument at a time to a function

Chapter 13 - Ajax

Asynchronous JavaScript XML - communicating with external APIS


Fetch API - Ajax requests


Response Interface - allows to control response received from a request


Request Interface - allows to create a request object that contains info about the request being made i.e. URL and headers


Headers Interface - allows HTTP headers to be added to a request or response object


Requests: GET (retrieve data) or POST (send data)

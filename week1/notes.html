Mobile UX article: Mobile UX design

MOBILE UX

- design of UX for hand-held & wearable devices

Mobile User Scenarios:

  1. Microtasking - short spurt uses
  2. Local - around them
  3. Bored - surf while waiting
Goals of Mobile UX video notes:
Mobile is inherently one eyeball and one finger - Luke Wroblewski (pioneer in web ux design) - design with this in mind.

Goals of Mobile UX:

  • Easy to use - intuitive, quick performance
  • Easy to learn - short user learning curve
  • Satisfying - joy of use, more visual
  • Avoid Interface Friction - keep user feeling good; design should synch with users

How to Streamline Mobile UX:

  1. Decide design:
    > Responsive - adjusts to all devices
    > Adaptive - several versions per screen of various sizes
  2. Do's
    1. Minimize Content - minimal page-loading times
    2. Simplify Navigation - easy to use, short-key access, straightforward links
    3. Restrict User Inputs - offer max effect for min effort
    4. Ensure Continuity and Consistency with both content and design
